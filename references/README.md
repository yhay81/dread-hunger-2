# References

This directory is for manifests, notes, inventories, and ignored private research copies.

The research repo may keep local-only raw copies under `references/private/raw/` so they can be studied without depending on external folder locations. That directory is ignored by Git and must never be staged, committed, pushed, used as a production dependency, or migrated into the future product repository.

Tracked files in this directory should stay limited to:

- source path manifests
- copy manifests
- research workflow notes
- original summaries and high-level operational lessons

Do not commit Dread Hunger server binaries, packaged game content, assets, executable files, databases, personal logs, or copied implementation details into Git history.
