# Build And Upload Notes

Steam build upload is out of scope until Phase 2 or Phase 3. Keep this file as the checklist placeholder.

## Before First Steam Upload

- Steamworks partner setup complete.
- AppID created.
- Build depots planned.
- AI Content Survey draft updated from actual shipped content.
- Store claims match implemented features.
- No unlicensed placeholder assets in the build.
- Dedicated server depot decision made.

## Build Types

| Build | Use |
| --- | --- |
| LocalDevelopment | internal testing |
| PlaytestCandidate | Steam Playtest upload |
| DemoCandidate | public demo upload |
| EarlyAccessCandidate | release review |
| DedicatedServerTool | Steam Tool App for official and community hosting |
