## Context

The project carries four names across layers: public/working title **Frostwake**, UE
codename/module **AbyssLock**, repo path **dread-hunger-2**, and the reference game **Dread
Hunger** (not ours). The Rust crate is already `frostwake-tools` and
[AbyssLock.uproject](../AbyssLock.uproject) already *describes* the game as Frostwake, so the
laggard is the `AbyssLock` UE module + `Abyss*` C++ class prefix (910 occurrences / 169 files).
Owner decision (2026-05-30): do a **full rename** of the UE module **and** the C++ class prefix
to Frostwake; **defer** the repo/folder + GitHub-remote rename.
