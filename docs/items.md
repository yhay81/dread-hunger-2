# Items

## 初期10アイテム

| Item | 用途 | 面白さ |
| --- | --- | --- |
| 防寒外套 | 船外で低体温の進行を抑える | 単独探索の誘惑と外套の奪い合いを作る |
| ヘッドランプ | 停電区画や吹雪前後の視界確保 | 便利だが光で位置が目立つ |
| 工具巻き | エンジン、発電機、ハッチ、ポンプの基本修理 | 誰でも使うため所在確認の材料になる |
| 応急包帯 | ダウン/負傷の応急処置 | 救助行動に証言と疑いが生まれる |
| 魔法瓶 | 短時間だけ低体温を回復 | 船外探索の帰還判断を作る |
| 信号弾 | 船外で位置通知、救助要請、陽動に使う | 本物の救助か罠かを疑わせる |
| アイスツール | 氷上移動、簡易近接、防寒区画の開放 | 武器にも道具にもなる |
| 方位データカード | 航路進行と船外探索の帰還補助 | 紛失・改ざんが疑いになる |
| 通信モジュール | 無線復旧に必要 | 終盤の勝利進行物 |
| 小型燃料缶 | 発電または航行を少し進める | 汚染/持ち逃げ/隠匿の疑いを作る |

## 実装ルール

- アイテムは原則として1人1-2個まで。
- 重要アイテムは位置ログを残すが、全員に常時公開しない。
- 「誰が最後に持っていたか」が試合後に話題になる設計にする。
- 初期白箱ではクラフトを入れない。
- 料理、食人、人肉、呪術系アイテムは入れない。
