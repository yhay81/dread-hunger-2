# Cultural Representation — 先住民・他文化の描写方針（v0.1）

> 確定ベース: 北極 c.1910年代（[`setting-bible.md`](setting-bible.md)）。北極は **先住民の土地**。本作は「氷を知る者の
> 知が軽んじられる」テーマ（史実: 北極先住民の知の黙殺）を扱いうるが、これは **倫理的・IP的に最も慎重さを要する**
> 領域。本書はその方針。**公開前に必ず sensitivity review を通す**（`docs/ip-boundary.md` Development Phase Policy）。

## なぜ慎重にやるか
- 「神秘の原住民が悪霊を察知する」式は **流用＋ステレオタイプ**で、しばしば侮辱的。
- the Pale を実在文化の精霊観に重ねると **聖性の盗用**になる。
- 一方、テーマ自体（傲った遠征が地の知を黙殺して滅ぶ）は強く、**正しく描けば価値が高い**。

## 5原則
1. **既定は「架空の北の人々」。** 実在民族（イヌイット/ユピック/チュクチ等）の **固有名・聖性・具体的文化を
   流用しない**。架空の沿岸の人々を創作し、実在を写さない生活・言葉・地名を与える。
2. **the Pale は我々の創作であり、実在の信仰ではない。** 蒼の宇宙論を **実在文化の神話・精霊観に重ねない**。
3. **知は霊感でなく実用。** 案内人・沿岸の人々の知は **氷・天候・白熊・「あの方位へ行くな」という観察と警告**。
   彼らが正しいのは超能力ゆえでなく **土地を知るゆえ**。
4. **主体性と内面。** 道具・犠牲・「高貴な野蛮人」「神秘の助言者」にしない。名・動機・弱さ・ユーモアを持つ個人。
5. **悲劇の責は遠征側に。** 「忠告を無視した愚かな探検隊」は **帝国の傲り** として描き、
   「神秘の原住民 vs 文明」の対立図式にしない。

## 案内人（the guide）の設計指針
- 役割: 氷・熊・航路・「the Bearing は死へ向かう」を知る、沿岸の人。乗員にも沿岸の助力者にもなりうる。
- **やってよい**: 実用的な助言・警告・生存術、（架空の）独自の信念体系、遠征への懐疑、個としての関係。
- **やらない**: the Pale との霊的交信／予言／超能力／聖性の道具化／実在民族の写し。
- 名・人々の名は **sensitivity review まで〔保留〕**（[`dramatis-personae.md`](dramatis-personae.md) / [`names-and-glossary.md`](names-and-glossary.md)）。

## Dos / Don'ts

| やる | やらない |
| --- | --- |
| 架空の北の人々を創作し、実在を一対一で写さない | 実在民族名・聖地名・実在の儀礼/意匠を流用する |
| 知＝土地の観察と警告として描く | 「悪霊を察知する霊能者」にする |
| 個としての内面・動機・関係を与える | 道具/犠牲/背景の飾りにする |
| 遠征側の傲りを悲劇の主因にする | 「文明 vs 未開」の構図にする |
| 不安なら案内人・沿岸描写を最小化できる設計に | 物語を彼らの神秘性に依存させる |

## プロセス
- 公開前ゲートに **sensitivity reader レビュー** を必須で含める（発注パケット: [`sensitivity-review-brief.md`](sensitivity-review-brief.md)）。
- 架空の人々の言語・地名・意匠が特定実在文化と一対一にならないよう監修。
- **依存しすぎない設計**: 案内人/沿岸の人々を縮小・撤去しても物語が成立するよう作る（保険）。
