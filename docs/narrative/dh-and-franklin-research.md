## 1. フランクリン遠征（public domain の史実 — 自由に汲める源泉）

DH も我々も、この実話から汲む。**史実・事実は著作権の対象外**（特定著者の文章表現を写さなければ自由）。
以下は複数の公開情報源で確認した事実の要約（出典は §4）。

### 航海と船
- **1845年5月19日**、英ケント州 Greenhithe を出航。2隻・**総勢134名**（士官24＋兵110）。
  グリーンランドで結核の5名を帰還させ、以後 **129名**。
- 2隻は探検用に改造された頑丈な船で、**帆＋補助蒸気機関**（鉄道機関車を転用した蒸気機関＋
  スクリュー1軸、氷から守るため引き上げ可能）、**鉄板で補強した船首**、**船内蒸気暖房**、
  **3年分の食料（缶詰スープ/野菜・塩漬肉・ペミカン・生牛）**、千冊超の蔵書を積んだ。
  → **「鉄帯補強の木造耐氷蒸気帆船＋石炭＋ボイラー＋缶詰」という我々の設定は史実に裏打ちされる。**
- 缶詰は出航7週前に契約した業者製で、**鉛の半田が雑**だった（後の鉛中毒説の根拠）。

### 破滅
- 欧州人による最後の目撃は **1845年7月末**（Baffin 湾で捕鯨船が遭遇）。以後消息不明。
- **1846年9月12日**、King William 島沖の Victoria 海峡で結氷に閉じ込められ、二度と動かず。
- **Victory Point の記録**（発見された唯一の文書、海軍定型用紙の二部構成）:
  - 第一部（1847年5月）「**All well**（万事順調）」。
  - 第二部（**1848年4月25日**）: **4月22日に2隻を放棄**。**フランクリンは1847年6月11日に死亡**。
    この時点で **士官9＋兵15＝24名死亡**。残る **105名** が Crozier 指揮で **南の Back River へ徒歩**。
  → **「最初は順調 → 後から死者数だけが書き足される」未完の記録**は、両義型・証拠系に転用できる装置。
- 生存者は全滅。30〜40名が本土北岸に達したが、なお数百マイル手前で力尽きた。

### 死因（＝合理的説明の層）と人肉食
- 寒さ・飢餓・**壊血病**・**鉛中毒**（近年は生涯通じ一定で「急性中毒」説に反証も）・
  **亜鉛欠乏→免疫低下→結核**・肺炎が複合。**幻覚・錯乱・猜疑に医学的説明がつく** = 両義型の土台。
- **人肉食は史実**: 骨の解体痕・煮沸痕（1992-93調査）、Inuit 証言（1854 Rae 報告）、
  2024年に Fitzjames の下顎骨の切創を DNA同定で確認。
  → ただし **人肉食は DH も濃く使い、現行 art-bible は禁止**。史実だが「DH 由来に見える危険」がある
  「ホットゾーン」。採否は要判断（§3 / 後述の問い）。

### Inuit 証言と捜索遠征（テーマの源泉）
- **Inuit の証言は正確だった**のに、ヴィクトリア朝の英国社会は長く黙殺した（人肉食報告に世論が反発）。
  → 「土地を知る者の知 vs 帝国の傲り」「信じられなかった真実」というテーマは、独自に翻案する価値大。
- 大規模な捜索遠征が続いた（Rae 1854、McClintock 1859 が記録発見、Hall、Schwatka …）。
  → **「消えた先行隊を捜す後続遠征」**という枠は史実そのもの。我々の航海フック（先行隊＋信号）の正統な下敷き。
- 沈没船は **Erebus が2014年・Terror が2016年** に、Inuit の伝承通りの場所で発見。

## 2. Dread Hunger（保護された表現 = 避ける境界）

以下は **写さないための境界線**。公開情報に基づく高レベルの要約（我々の言葉で記述。DH のロア文・
台詞・アセットは引用・複製しない）。**機構は写してよいが、ここに挙げる固有表現は避ける。**

- **舞台**: 1840年代の北極、北西航路。8人の乗員（うち2人が裏切り者）が蒸気船で氷海を踏破する。
  **フランクリン遠征に明示的に着想**し、そこへ **超自然（精神を蝕む北極の存在・血の魔術儀式）** を上乗せ。
  → 史実部分は我々も自由に使える。**「超自然の上乗せ方」こそ DH 固有の創作**で、ここを写さない。
- **避けるべき DH 固有表現**（複製禁止。プレースホルダにも使わない）:
  - 役職名の **セット**（DHは Captain/Navigator/Engineer/Cook/Doctor/Chaplain/Royal Marine/Hunter）。
    一般階級語の単独使用は可だが、**この8人構成をなぞらない**（我々の roster は独自設計 — `roles.md`）。
  - 裏切り者/乗員の **呼称**（DH の "Thrall"/"Explorer"）。→ 我々は「乗員 / 工作員（operative）」。
  - **船名**（DH 固有名）。→ 我々は **Frostwake**。
  - **超自然システムの具体表現**: 血の魔術・呪文・召喚する怪物・精神汚染エンティティの「DH での設計」。
  - **署名級の裏切り機構**（`mechanics-parity-target.md` pillar 11-12）: 人肉食で力を得る/トーテム型
    設置物/瀕死者を「下僕化」する変換 — **DH の設計のまま写さない。独自設計＋IP Review 必須**。
  - **最終目的の具体表現**（DH: ニトログリセリンで巨大氷山を爆破して脱出）。→ 我々は最終航行/救助航路。
  - DH のマップ/アート/UI/音/ロア文。

## 4. 要判断フォーク（このあとの問いで確定）

研究の結果、**未解決の本質的フォークが2つ**残る。次の質問で確定する。

1. **超自然の在り方と裏切り者の power fantasy。**
   - DH は **顕在型**（裏切り者が実際に魔術/超自然パワーを使い、超自然は実在として確定）。
   - だが前回オーナーは **両義型**（超自然を決して確定しない）を選択。**両者は噛み合わない。**
   - 「システム＝DH」を文字通り取ると顕在型に寄る。両義型を取ると DH の裏切り者パワーから離れる。
     → **(A) 両義型** ／ **(B) 顕在型だが完全オリジナルの超自然** ／ **(C) ハイブリッド（パワーはあるが両義に演出）**。

2. **年代の精度。**
   - **1840年代（フランクリン期＝DH 準拠）**: 最も DH/史実忠実。ただし **無線も電灯も無い**
     → 「無線」は信号灯/狼煙/ケルン書置き、「発電」は油灯＋ボイラー火に要リスキン。
   - **英雄時代 c.1900-1910s**: 蒸気帆船のまま **無線電信・電灯が史実的に成立** → 既存の
     発電/無線メカニクスを **ほぼ無改修で温存**。DH の正確な年代からは少し離れる。
