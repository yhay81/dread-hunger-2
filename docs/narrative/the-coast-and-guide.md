## 設計の核（5原則の適用）

- **既定 = 架空の沿岸の人々。** 実在民族（イヌイット/ユピック/チュクチ/サーミ等）を **写さない・名指さない**。
  実在の自称・聖性・意匠・言語形態素を流用しない（"-vik/-miut" 等の実在風語尾も避ける）。
- **モノリスにしない。** 「部族」ではなく、この海岸で世代を重ねた **生活者の共同体**（猟（海獣）・罠・交易）。
  個人として描き、ラベルで束ねない。
- **the Pale は彼らの宗教ではない。** 蒼を彼らの精霊観・神話に重ねない。彼らの信念体系は（描くなら）**架空**で、
  the Pale とは無関係。
- **知は実用。** 氷・天候・海獣・白熊・季節・「あの方位（the Bearing）へ行く船は還らない」という **観察と経験則**。
  超能力でも predes予言でもない。
- **悲劇の責は遠征側に。** 彼らの警告を **Society の傲り**が退ける——「文明 vs 未開」ではなく「現地の知 vs 帝国の慢心」。

## 沿岸の人々（the coast community）

- **居所**: Mercy Point（沿岸の交易所/伝道所）とその後背地に点在する猟・罠の集落。遠征の「遠い救い」。
- **生業と関係**: 海獣猟・毛皮・交易。捕鯨船や交易者と何世代も取引してきた——だから遠征隊に対して
  **従属でも無垢でもなく、対等で現実的**（値段交渉もすれば、危険な依頼は断る）。
- **自分たちの利害**: 彼らには彼らの冬支度・家族・損得がある。遠征の救助は **彼らの最優先事ではない**。
  助けるかどうかは取引と信頼の問題。→ 物語が彼らの善意に甘えない。
- **多様性**: 一枚岩でなく、遠征に懐疑的な者、商機を見る者、関わりたくない者がいる。

## 案内人（the guide）— 一人の人間

- **立ち位置**: 沿岸の人々の一人。捕鯨船で働いた経験があり、Society の言葉も操る——**世界の境界を渡る人**。
  乗員として同乗するか、Mercy Point で助力する側か（どちらでも可。物語が依存しすぎない設計）。
- **知**: 氷況・天候・白熊の習性・航路・「the Bearing の先からは誰も還らない」。これは **経験と観察**であって霊感ではない。
- **主体性とアーク**: 自分の目的（報酬/家族/借り/好奇）を持つ。**正しいのに退けられる**痛み、続けるか降りるかの選択、
  乗員との対等な摩擦と信頼。怯えも皮肉も冗談も持つ **完全な人物**。
- **the Claimed との関係**: 案内人は the Pale と霊的につながらない。だが「氷に呼ばれた者」を **経験則で見抜く**ことは
  ありうる（"あの目つきは知っている"——民俗知であって超能力ではない）。これも断定させず、誤りうる人間の見立てとして。

### 案内人で「やらない」（the guide must NOT be）
- the Pale/蒼との霊的交信・予言・超能力・聖性の道具。
- 乗員（プレイヤー）のためだけに存在する助言装置／犠牲／背景の飾り。
- 「高貴な野蛮人」「神秘の老賢者」「最後に正しさを証明して死ぬ原住民」テンプレ。
- 実在民族の写し・実在地名/聖地/儀礼の流用。

### 台詞・描写の原則
- 実用と具体（「ここの氷は若い。橇は通すな」）。神託調にしない。
- 訛りや片言で **知性を低く見せない**。多言語を渡る有能さとして描く。
- 内面（家族・損得・恐れ・ユーモア）を地の文/会話で見せる。

## ゲーム上の扱い（依存しすぎない設計）
- 案内人は **乗員8人の1枠にも、沿岸の NPC 助力者にも**できる（[`dramatis-personae.md`](dramatis-personae.md) の「案内人」枠）。
- **最小化・撤去の保険**: レビューで懸念が出たら、案内人を NPC 助力者に縮小、または沿岸描写を「交易所と取引」だけに
  絞り、物語の骨格（the Pale/the Claimed/carve-free）は無傷で成立する設計にしておく。

## Self-audit（自己監査 — 人間レビューの前段。代用ではない）

| 監査項目 | 結果 |
| --- | --- |
| 実在民族を写す/名指す/聖性を流用していないか | ✅ 設計上 無し（架空・非マッピング。語尾も実在風を回避） |
| the Pale を彼らの信仰に重ねていないか | ✅ 無関係と明記 |
| 知を霊感でなく実用として描いているか | ✅ 経験則として設計 |
| 主体性・内面・対等性を与えているか | ✅ 利害/家族/懐疑/拒否権を付与 |
| 「文明 vs 未開」構図／ステレオタイプ回避 | ✅ 責は遠征側／テンプレ禁止を明記 |
| ステレオタイプな台詞・訛り表現の回避 | ✅ 原則を明記（実装時に再点検） |
| 依存しすぎない（縮小/撤去可能） | ✅ 保険を設計 |
