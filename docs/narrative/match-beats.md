## 試合アーク（6ビート・再掲）
1. 出航と日常 → 2. 見捨てと最初の異変 → 3. 猜疑の発火 → 4. 進路の対立（the Bearing vs 脱出/Mercy Point）→
5. クライマックス（氷から船を解き放つ vs 明け渡し）→ 6. 語りたくなる余韻。

## 事件の種（library）
各種: **引き金 / 蒔く疑い / 残る痕跡 / 生まれる会話**。the Claimed の力（Quench/Rime/Mar/Call/Mark/Deliver）と対応。

## 運用指針（vision の成功判定に対応）
- **1試合 最低1つ**の「語れる事件」。**2-4分に一度**の意味ある遭遇/会話。
- 妨害は **推理材料を残す**（理不尽でない＝§7「痕跡」）。隠れ得が支配しないよう検知リスク調整（GP-03）。
- 種は **両義の仮面**を必ず併走（顕在型でも即断不能）。1試合に超自然色を盛りすぎない（多くは "たぶん寒さ"）。
