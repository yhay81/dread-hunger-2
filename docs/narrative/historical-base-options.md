## 選定基準（ゲーム適合）

DH 型ループに載るかで評価する:

1. **船ループ**: 氷に囚われた船を **前進/解放させ続け、climactic に脱出** できる形か（DH の核）。
2. **閉鎖乗員**: 8人規模に抽象化できる閉じた集団か。
3. **寒・孤立**: 低体温・吹雪・暗闇の圧があるか。
4. **裏切り適合**: 隠れた工作員/内紛が **自然に成立** する素地があるか。
5. **顕在型(the Pale)適合**: 闇・狂気・氷の「招き」を **重ねやすい** か。
6. **Franklin/DH からの新鮮さ**: DH と差別化できるか（= IP に効く）。
7. **public domain**: 史実(=事実は自由) か、著作権切れ古典か。著作権作品は **ジャンル着想のみ**(筋・人物・固有名は不可)。

## 推奨

**主軸 = Belgica（1897-99・南極・史上初の越冬）** を土台に、**裏切りの力学を Karluk（見捨て・内紛・殺害）から
移植**し、**超自然は独自の the Pale**。理由:

- **船ループに構造ごと合う**: 「氷に閉じ込められた船を、暗闇と消耗の中で **自力で解き放ち脱出する**」が
  そのまま 1試合。クライマックスの **氷に水路を掘削・爆破して船を解放** は **史実**（→ DH のニトロ氷山爆破に
  寄せずに済む正当な根拠＝IP に有利）。
- **顕在型(the Pale)に最適**: **極夜＝闇の時間** は「闇が入り込む/蒼が招く」を載せる理想の器。史実の
  ポーラー・マッドネス(壊血病・極夜鬱)が「正気を失う合理的理由」＝§7「寒さの仮面」を常に供給。
- **新鮮で IP に効く**: **南極**は Franklin/The Terror/北氷洋(いずれも北極)から明確に離れる。DH の土台
  (Franklin)を踏まない。
- **裏切りは付加で自然**: 史実の Belgica は精神崩壊が主で sabotage ではない。が、闇で正気が削れる越冬船に
  **隠れた the Claimed が2人** 紛れる構図は完全に自然（= オーナー言う「ゲームに合うよう史実を調整」）。
  Karluk の **見捨て・派閥・殺害** を力学として注入する。

### ベース別の航海フックの違い（参考）
- **Belgica/南極**: 「最初に闇へ踏み込む隊。氷に囚われ、極夜が the Pale を連れてくる。解き放つか、獲られるか」。
  先行隊フックは任意（南極初越冬=史実上は先行隊なし。欲しければ架空の先行隊を足す）。
- **Karluk/北極**: 「捜索/調査の隊が氷に囚われ、頼みの指揮が消える(=見捨て)。乗員が割れる」。
  消えた先行隊(the Perdita)フックは北極の捜索遠征伝統に自然に乗る。

## ゲームへの「フィット調整」方針（システムは不変）

選んだベースは、DH 同型ループに合わせて以下を調整:

- **船を抽象化**: 史実の船 → フロストウェイク号（帆＋補助蒸気の耐氷船）。8人/2 the Claimed に編成。
- **「前進」= 氷からの解放**: 漂流/閉込の史実を「氷況帯を抜け、最後に水路を切り開いて脱出」に翻案。
- **既存メカ温存**: 石炭・ボイラー・暖房・灯(油灯/炉)・信号(信号灯/書置き)・航法。妨害は the Pale の力で再演出。
- **固有名は完全オリジナル化**: 実在の船・隊・人物名は使わない（Belgica/Karluk 等は **着想元**。
  我々は Frostwake / the Meridian Society / the Perdita / the Pale）。
- **年代の手触り**: 主軸 Belgica なら **1890年代末**（電灯黎明・無線はまだ＝灯は脆い炎、連絡は信号灯/書置き）。
  Karluk なら **1910年代**（無線が載り始める＝中心の謎を無線に絡めることも可能）。
