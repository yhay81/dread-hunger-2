# IP Boundary

## Reference Boundary

Reference only the emotional structure and broad genre:

- hostile ice environment
- shipboard cooperation
- proximity voice suspicion
- scarce resources
- 1-2 hidden saboteurs
- short match stories

Do not reference protected expression:

- names, logos, UI, characters, roles, item names, map layouts
- occult systems, blood rituals, totems, cannibalism, human meat
- same ship silhouette, same room composition, same final objective
- store copy or marketing that implies affiliation

## Product Identity

Frostwake is:

- near-future fictional icebound research and response vessel
- diesel-electric / battery-assisted metal ship
- agents/saboteurs rather than magical traitors
- fuel, hull, radio, route, power, heat, hypothermia
- Steam-native server browser and dedicated server hosting

Frostwake is not:

- a sequel
- a remake
- an unofficial continuation
- a clone marketed by another game's name

## Store Language

Safe direction:

- "icebound cooperative survival deduction"
- "proximity-voice polar survival"
- "crew-based betrayal survival"
- "keep the ship moving while someone sabotages the voyage"

Avoid:

- "Dread Hunger-like"
- "for Dread Hunger refugees"
- "spiritual successor"
- "unofficial sequel"
- "same as..."
