# Vision

作業タイトル: **Frostwake**

商標調査前の仮称。公開前に必ず名称調査を行う。`Dread`, `Hunger`, `Thrall`, `Expedition`, `Franklin` など強い連想語は使わない。

## 1ページ企画書

近未来の架空極地。氷海研究船フロストウェイク号は、氷海の奥から届いた救難信号を追って北方航路へ入る。乗員は燃料、船体、発電、無線、航路、低体温、吹雪に対処しながら船を進める。しかし乗員の中には、船を目的地へ到達させたくない工作員が紛れている。

狙う面白さは「船を進めるには協力が必要だが、協力相手を信じ切れない」。残すのは氷海の船上で起きる協力、疑心暗鬼、近接VC、物資不足、船外探索の心理体験。変えるのは時代、船型、裏切り者の設定、勝利条件、妨害方法、UI、マップ構造、アートスタイル、料理/食人/魔術要素。近未来化により、船内センサー、電力系統、研究区画、ドローン/ROV格納、非常LED、サーバー/無線ラックを使って独自表現を作る。

## ピラー

1. **近接VCでしか生まれない疑心暗鬼**
   声が届く距離、壁、扉、機械音、吹雪、船外通信ノイズで情報が欠ける。全体会議ではなく、現場証言、最後に聞こえた声、扉越しの交渉で人間関係が壊れる。

2. **船を維持するサバイバル**
   主題は飢えではなく、船体、エンジン、燃料、発電、無線、航路、低体温、視界、浸水、氷圧。船が止まれば全員が追い込まれる。

3. **PvPvEの圧力**
   工作員だけでなく、吹雪、低体温、船体損傷、燃料切れ、発電停止、無線障害、船外の脅威が同時に進行する。

4. **20分で語れる事件が起きる**
   毎試合、偽修理、燃料汚染、航路偽装、置き去り、船外で途切れる声、救助の失敗など、試合後に話したくなる場面を作る。

## 試合ループ

1. 船の状態を確認する: 船体、燃料、発電、無線、航路、低体温、氷圧。
2. 修理、船外探索、物資運搬、航路確認を分担する。
3. 近接VCで目撃情報、作業結果、疑いを共有する。
4. 工作員が燃料汚染、発電妨害、無線妨害、航路偽装、ハッチ凍結で混乱を作る。
5. 乗員は船を進め、氷圧帯を突破し、救難信号地点または海峡出口へ到達する。
6. 終盤は最終航行との競争になり、誰を信じて誰を船外へ出すかを決める。

## 勝利条件

### 乗員側

- 航路進行を完了する。
- エンジン、発電、無線、船体の最低状態を維持する。
- 最終航行シーケンスを完了し、生存者の規定人数が船内に残る。

### 工作員側

- 時間切れまで最終航行を阻止する。
- 乗員を規定人数以下に減らす。
- 船体崩壊、燃料枯渇、航路喪失、無線喪失のいずれかで船を致命状態にする。

## 最小版

- 通常マッチは8人固定。8人集まったらロビーから開始する。
- 練習プレイは1人で開始できる。
- 工作員は2人。
- マップは近未来の氷海研究船と氷上探索エリア1つだけ。
- 試合時間は白箱で20-35分、製品目標は18-25分。
- 近接戦闘中心。銃は初期実装しない。
- AI敵または環境脅威は1種類だけ。

## 残す体験 / 変える表現

| 残す体験 | 新しい表現 |
| --- | --- |
| 船が進まないと全員が死ぬ | 氷圧帯、燃料切れ、救難期限で追い込まれる |
| 物資や修理の妨害 | 燃料汚染、発電停止、無線周波数ずらし、航路偽装 |
| 船外探索の危険 | 吹雪、低体温、白out、方位喪失、通信ノイズ |
| 少数の裏切り者が情報優位 | 工作員同士の暗号サイン、偽報告、隠し目標 |
| 密談、叫び声、発見証言 | 距離減衰VC、壁越し減衰、船外ノイズ、無線途切れ |

## 30日白箱の成功判定

- テスト後に自然に「もう1回」が出る。
- 1試合に最低1つ、参加者が語りたくなる事件が起きる。
- 2-4分に一度、意味のある遭遇や会話が起きる。
- 工作員勝率が40-60%に入る兆しがある。
- 妨害が理不尽ではなく、推理材料を残す。
- 単独行動が便利だが危険という判断が発生する。
