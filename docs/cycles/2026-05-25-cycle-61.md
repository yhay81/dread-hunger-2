## Evidence

- `Tools/windows/check_steam_dev_config.ps1` fails if committed defaults switch to Steam, include `SteamDevAppId`, or enable `OnlineSubsystemSteam`.
- The same script can validate an ignored local `Saved\Config\steam_dev.local.ini` with `DefaultPlatformService=Steam`, `bEnabled=true`, `SteamDevAppId`, and `GameServerQueryPort`.
- `docs/steam-dev-config-gate.md` records the exact Windows commands for default and local-config checks.
