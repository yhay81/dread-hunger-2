## Parallel execution model (race-free)

The UE build and the git working tree are inherently serial (parallel writers caused half-commits
before). So multi-agent parallelism was applied to the **parallelizable** part only: three
**read-only** design agents ran concurrently and returned text (no repo writes); the main loop then
**integrated serially** (one set of edits, one build, one commit). Agents:
- **A (GP-08):** buildable ship-interior layout spec → `docs/gp08-ship-map-spec.md`.
- **B (GP-03):** survival-restore implementation plan → implemented this cycle (below).
- **C (GP-09):** JP string mapping + OFL font import plan → `docs/gp09-jp-localization-font-plan.md`.
