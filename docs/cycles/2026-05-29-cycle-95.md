## Planned Scope (Path A — builds today, engine materials)

- `CreateWhitebox` (`Source/AbyssLockEditor/Private/AbyssWhiteboxCommandlets.cpp`): rewrite the `Cubes`
  array — the old solid room blocks (e.g. `WB_Bridge` was a 620×520×260 filled cube) become a deck
  plate + outer hull shell + interior ceiling + a segmented central **spine corridor** + **hollow
  rooms** built from 40 cm wall segments with doorway gaps: Bridge / Radio (fwd), Quarters / Infirmary
  (mid), Engine / Battery (aft), full-width Fuel bay, and an open **bow ice deck** under the HDRI sky.
- Required labels (locked by the validators) ride on each room's representative wall — `WB_Hull_MainDeck,
  WB_Bridge, WB_RadioRoom, WB_Infirmary, WB_QuartermasterStore, WB_EngineRoom, WB_FuelBay,
  WB_BatteryRoom, WB_IcePressureGate` — all preserved; task labels/systems/modes and door ids unchanged.
- Repositioned PlayerStarts (spawn cluster in the open central gap), ship tasks, target points, doors,
  props, and the cycle-94 survival items/heat sources into the new rooms.
- Added 8 **Movable interior point lights** (`WB_Light_*`, no bake) so the enclosed rooms read at
  runtime (the arctic sun only reaches the open bow deck).
- Fixed a stale validator: `EditorStartupMap`/`GameDefaultMap` are now checked against `L_MainMenu`
  (the cycle-85 boot map) instead of the whitebox — the old check predated the dedicated menu boot.
