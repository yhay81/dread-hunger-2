## Background (assessment)

The shared objective across all modes is "advance the Route to 100% → Final Approach → crew win".
Confirmed gaps that stop solo being a complete 30-min game: (#1) the single Route task adds +0.35 with
no cooldown, so 3 presses win in ~10s and the 25-min timer never matters; (#2) no pacing/fuel gate;
(#3) no win/lose result screen (data is replicated but not shown); (#4) a downed solo player never
ends the match (crew-elimination loss required ≥5 players). This cycle fixes **#4** (model-agnostic)
and writes the plan for #1/#2/#3. Pacing model (A voyage / B checkpoints / C gated-manual) awaits
owner confirm — spec is drafted with **A (voyage)** recommended.
